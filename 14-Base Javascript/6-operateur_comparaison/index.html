Les opérateurs de comparaison en JavaScript

Opérateur Description Exemples qui renvoient true
Égalité (==) Renvoie true si les opérandes sont égaux après conversion en valeurs de mêmes types. 3 == variable1

"3" == variable1

3 == '3'
Inégalité (!=) Renvoie true si les opérandes sont différents. variable1 != 4
variable2 != "3"
égalité de type en JavaScript. 3 === variable1
Inégalité stricte (!==) Renvoie true si les opérandes ne sont pas égaux ou s'ils ne sont pas de même type. variable1 !== "3"
3 !== '3'
Supériorité stricte (>) Renvoie true si l'opérande gauche est supérieur (strictement) à l'opérande droit. variable2 > variable1
"12" > 2
Supériorité ou égalité (>=) Renvoie true si l'opérande gauche est supérieur ou égal à l'opérande droit. variable2 >= variable1
variable1 >= 3
Infériorité stricte (<) Renvoie true si l'opérande gauche est inférieur (strictement) à l'opérande droit. variable1 < variable2
"2" < "12"
Infériorité ou égalité (<=) Renvoie true si l'opérande gauche est inférieur ou égal à l'opérande droit. variable1 <= variable2
variable2 <= 5
